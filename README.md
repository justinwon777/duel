# Duel
A simple 2D space shooter created with PyGame. Play against an AI enemy or another human.

![](images/mainmenu.jpg)

![](images/ingame.jpg)

## Controls
- Player 1 (blue): WASD to move, spacebar to shoot
- Player 2 (red): Arrow keys to move, right alt to shoot
